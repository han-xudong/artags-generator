# Aruco 和 AprilTag 标签生成器

这是一个基于React和Vite构建的Web应用，用于在线生成、自定义和导出Aruco和AprilTag标记标签。

## 功能特点

- 支持生成Aruco和AprilTag两种标记
- 可自定义标签类型、字典/家族、ID、尺寸等参数
- 实时预览生成的标签
- 导出格式：SVG、PDF和PNG
- 响应式设计，适配各种设备

## 技术栈

- 前端框架：React 18
- 构建工具：Vite
- UI库：Material-UI (MUI)
- 状态管理：Zustand
- 导出功能：jsPDF、html-to-image、file-saver
